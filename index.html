BeyondWords Player

This is an example page that contains the BeyondWords Player. It is used during the development of the player to test the effects of code changes in this repository.

You can use this page to test different styles of player by running the ./bin/server script on your computer and then editing the index.html file in the root of this repository. The player is intialized in the script tag at the top of this page and takes various options.

For example, you could pass the playerStyle: "small" option to change the style of player. You can also make changes in a JavaScript console, without having to run the project locally, e.g. BeyondWords.Player.instances()[0].playerStyle = "small"

The automated tests in this project also use this page. They take screenshots of the player's user interface and compare them against previous images for regressions.
